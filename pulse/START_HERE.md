## ⚡ Test en 30 Secondes / 30-Second Test

### 1. Ouvrez Xcode
```bash
open pulse.xcodeproj
```

### 2. Changez la langue
- **Product** → **Scheme** → **Edit Scheme...**
- **Run** → **Options** → **App Language** → **French**
- Cliquez **Close**

### 3. Lancez l'app
- Appuyez sur **⌘R**
- 🎉 **Tout est en français !**
